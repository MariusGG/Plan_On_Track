### Installation

- Step 1
  ````
  $ Brew install yarn
  ```
- Step 2
    ````
    $ yarn add expo-cli
    ```  
- Step 3
    ````
    $ yarn install
    ```
-  Step 4
  ````
    $ yarn start
  ````
